# Mystery of Ancient Darkness: Initium

The complete illustrated novella is the canonical literary prelude to the
game. It establishes the characters and events that the visual prologue
condenses and the game continues.

- `MOAD_COVER_1.png`: Part I, *The Comptons*.
- `MOAD_COVER_2.png`: Part II, *The Journey*.
